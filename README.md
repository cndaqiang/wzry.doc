## Introduction
This repository contains a template for an open-source software manual, built with [MkDocs](https://www.mkdocs.org/) using the [Material theme](https://squidfunk.github.io/mkdocs-material/). It includes features such as visitor statistics, social media integration, common scripts, and deployment to [Read the Docs](https://readthedocs.org/).
