## 王者荣耀礼包
若需调整第`mynode`个账户领取哪些礼包

* 则在运行目录,创建`WZRY.mynode.运行模式.txt`文件
* **把`mynode`替换为你的[账户编号](config.md#mynode与instance的区别)**
* 文件为UTF8格式编码, 内容为标准的python语法,不支持超过一行的python语句.


**礼包功能**

* 在`WZRY.mynode.运行模式.txt`中**填入下面的内容来开启指定礼包**
* 礼包是三小时领取一次，或者当天的对战时间结束后才领取。**不是你开了功能就能立刻领取的**。
* 如果是调试代码，测试能否领取，额外填写一个`self.每日礼包(强制领取=True)`到`WZRY.mynode.运行模式.txt`的最后一行
* **个人开发者, 精力有限, 礼包功能不会随着游戏更新立刻进行升级维护, 也可能放弃维护.**
* **领取礼包遇到问题时, 欢迎[提出问题](https://github.com/cndaqiang/WZRY/issues), 但不保证给你解决**

```
# 礼包功能默认关闭,设置为True,开启礼包功能
self.启动礼包功能 = True
# 下面的礼包需要领取,则设置为True
self.每日任务礼包 = False
self.礼包功能_邮件礼包 = False
self.礼包功能_妲己礼物 = False
#
# 外置礼包,暂无手册,遇到问题,请自行调试
# 支持自动领取相同模拟器内部的王者营地礼包
self.外置礼包_王者营地 = False
# 支持自动领取相同模拟器内部的王者体验服的体验币
self.外置礼包_体验服 = False
# 精力有限, 以下礼包不会实时维护, 遇到问题,请自行调试
self.活动礼包 = False
self.祈愿礼包 = False
self.玉镖夺魁签到 = False
self.礼包功能_友情礼包 = False
self.友情礼包_积分夺宝 = False
self.友情礼包_皮肤碎片 = False
self.友情礼包_英雄碎片 = False
self.友情礼包_铭文碎片 = False
self.友情礼包_皮肤宝箱 = False
self.友情礼包_回城宝箱 = False
self.友情礼包_击败宝箱 = False
self.友情礼包_排位保护 = False
self.礼包功能_回忆礼册 = False
self.礼包功能_灵宝互动 = False
self.礼包功能_战队礼包 = False
self.礼包功能_商城礼包 = False
self.礼包功能_KPL礼包 = False
```

有些礼包没必要领取:

* **有时间限制的礼包, 比如`活动礼包`,`祈愿礼包`,`玉镖夺魁签到`,`礼包功能_KPL礼包`**, 这些活动很久才出现一次, 平时都保持为`False`节省时间.
* **没有性价比的礼包, 比如`友情礼包_铭文碎片`,`友情礼包_击败宝箱`**, 也建议设置为`False`
* **老用户没必要领的礼包, 比如`友情礼包_英雄碎片`,`礼包功能_战队礼包`**, 也建议设置为`False`
* 如果没有战队, 就没有必要设置`礼包功能_战队礼包`
* KPL赛事战令免费版在80级以上就没有礼包了, 就没必要设置`礼包功能_KPL礼包`了
* 如果没有友情币, 就没有必要设置`礼包功能_友情礼包`
* 友情币少的就只开最有性价比的`友情礼包_积分夺宝`和`友情礼包_皮肤碎片`
* 关于`每日任务礼包`是否要领取, 可以看[战令经验最大化](../exp/zhanling.md)


## 王者营地礼包
* 体验币换皮肤碎片、营地币换英雄碎片、战令经验包、签到、观赛等. 
* 领取过程展示:[王者营地日常任务合集和农活自动化助手拾取演示](https://www.bilibili.com/video/BV11WcgecEXF/)
* 可以独立启动、不依赖于`wzry.py`, 王者营地所在模拟器的ADB信息填到`config.ce.yaml`
* `config.ce.yaml`的配置规则同[配置文件](../guide/config.md)

```
python -u wzyd.py config.ce.yaml
```

### 控制文件
* 控制文件 `王者营地.mynode.初始化.txt`

* 老账户无需领取英雄碎片了, 可以关闭`营地币换碎片`
* 注册了体验服, 营地中可以`体验币换皮肤碎片礼包`
* 因此增加了两个开关控制

```
self.礼包功能_营地币换碎片 = True
self.礼包功能_体验币换碎片 = True
```


## 体验服礼包
* 体验服: 更新体验服领取体验币
* 可以独立启动、不依赖于`wzry.py`, 王者体验服所在模拟器的ADB信息填到`config.ce.yaml`

```
python -u tiyanfu.py config.ce.yaml
```

!!! danger "在Root的安卓上登录体验服,会被封号,与本脚本无关.如使用体验服的功能,请关闭模拟器的Root选项"
