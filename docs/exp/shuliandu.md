## 熟练度/经验获取
* 只有胜利才能获得熟练度, 所以必须采用挂机托管模式(`self.触摸对战 = False`)
* 青铜人机(`self.青铜段位 = True`)能提升到蓝色熟练度
* 星耀人机(`self.青铜段位 = False`)能提升到红色熟练度
* 刷熟练度使用快速模式就行(`self.标准模式 = False`)
* 刷熟练度建议单人模式进行(`self.组队模式 = False`), 毕竟还有一个*真人*玩家在打人机, 胜率高.



## 熟练度方案
> [**最新消息: 人机闯关胜率最高、对局最快,简直就是天生的刷熟练度圣体.** 但存在信誉分风险](../guide/duizhanmoshi.md#关于人机闯关模式的一些想法)

* **一般新号、新英雄都是用青铜人机**, 老号才考虑用星耀人机
* 先用**青铜人机**, 把所有英雄都刷成绿色熟练度, 就不受排位的限制了
* 如果有时间, 继续**青铜人机**把所有英雄刷成蓝色
* 星耀人机每天只有10次, 建议只刷常用英雄就行了, N年才能用人机对战把所有英雄刷成红色.
